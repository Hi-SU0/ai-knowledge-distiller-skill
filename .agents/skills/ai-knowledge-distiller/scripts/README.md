# Scripts

Version 0.1 is intentionally instruction-only. This directory contains no executable downloader, crawler, speech-recognition pipeline, or platform-bypass code.

## Possible future tools

- Vault inventory and source-count report.
- Import of user-provided or officially exported transcripts.
- Concept alias and duplicate-candidate checker.
- Obsidian wikilink integrity checker.
- Five-source maintenance report generator.
- Frontmatter and source-completeness validator.

Future scripts must default to non-destructive behavior, preserve human edits, emit a change log, avoid secrets, and require explicit authorization before moving or deleting files.

## Out of scope for v0.1

- Automatic Bilibili or YouTube video downloads.
- Circumventing access controls or platform restrictions.
- Automatic speech recognition.
- Unreviewed bulk ingestion.
